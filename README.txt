OPERAÇÂO
Cliente -> corre as funcionalidades do cliente
Para correr é nessesario criar um cliente no programa de servidor
Servidor -> corre as funcionalidades do servidor
Para correr utilizar o 
Username 'gestor'
Password (localizada na 1ª linha asseguir ao 1º ';' do ficheiro user.txt),por defenição password

Os programas utiliza no seu funcionamento os ficheiros de texto
user.txt -> Guarda todos os utilizadores
product.txt -> Guarda todos os produtos
gestor-stats.txt -> guarda as estatisticas gerais
Tambem cria outros fixeiros no formato:
username-stats -> Guarda as estatisticas para os produtos comprados para cada utilizador
username-LC -> Guarda a lista de compras do utilizador
Estes ficheiros NÃO devem ser manualmente alterados.
Uma vez que é utilizado o commando "System" os programas apenas correm em linux.
